# Collection Property Fill

**Type:** Global Plugin

Bulk-fill a text property on every note in an existing collection. The plugin adds a command palette command where you choose a collection, choose an existing text property or enter a new property name, then apply one value to every note in that collection.

## Features

- Choose any existing collection in the workspace
- Use an existing editable text property
- Create a new text property on the selected collection when needed
- Set the same text value on all notes in that collection
- Shows success/failure counts after the bulk update

## Installation

1. In Thymer, press `Ctrl+P` and search for **Plugins**
2. Create a new **Global Plugin**
3. Paste the contents of `customCode.js` into the Code tab
4. Paste the contents of `configuration.json` into the Configuration tab
5. Press **Preview**, then **Save**
