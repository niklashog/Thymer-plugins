# Dashboard

> ⚠️ **Work in progress** — this plugin is under active development and not yet functional.

**Type:** Global Plugin

A workspace-wide task dashboard that aggregates tasks from across your entire Thymer workspace and lets you manage them in one place.

## Planned Features

- Collect and display tasks from all collections in the workspace
- Unified task management view across projects
